# CircuitBuilder 3D - Landing Page

Modern marketing website for the CircuitBuilder 3D application.

## Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (Animations)
- **React Three Fiber** (3D Visuals)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure
- `src/components`: Reusable UI components (Hero, Features, etc.)
- `src/app`: Next.js pages and layout
- `public`: Static assets
